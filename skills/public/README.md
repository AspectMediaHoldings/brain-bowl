# Public Skills

This tier contains stable, domain-agnostic skills with no reuse restrictions.

The active public skill library lives in `.claude/skills/`. That directory holds 150+ pre-built skills covering scientific computing, document creation, data analysis, and more.

Add a skill folder here only when you have a general-purpose skill that does not belong in the team tier and is stable enough to share or reuse across contexts.
