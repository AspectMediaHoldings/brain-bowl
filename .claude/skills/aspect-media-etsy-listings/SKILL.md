# Dogs and Coffee — Etsy Listing Copy Skill

## Business Context

- **Business:** Aspect Media Holdings, LLC
- **Product line:** Dogs and Coffee
- **Platform:** Etsy (primary), Amazon Merch on Demand (future)
- **Production:** Printify (print-on-demand)
- **Current breeds:** Long Hair Chihuahua, Short Hair Chihuahua, German Shepherd, French Bulldog, Rottweiler, Yellow Labrador, Black Labrador
- **Design styles:** Minimalist, Line Art, Photorealistic
- **Saying categories:** Humor, Identity, Dogs+Coffee+No People

## Etsy Listing Structure

Each listing requires five elements: Title, Price, Tags, Description, and
Shipping/Size note. Produce all five for every listing requested.

---

## 1. Title

Etsy titles can be up to 140 characters. Etsy's search algorithm reads the
first 40 characters most heavily — put the highest-value keywords there.

**Title formula:**

`[Breed] Shirt [Saying Type] | [Design Style] Dog Mom Gift | Dogs and Coffee Tee`

**Rules:**
- Lead with the breed name — this is what buyers search first
- Include the saying or a short version of it naturally in the title
- Add "Dog Mom" OR "Dog Dad" OR "Dog Lover" — these are high-volume search terms
- Include the gift angle ("Gift for Dog Mom," "Dog Lover Gift") — Etsy skews heavily toward gift purchases
- Do NOT keyword-stuff with unrelated terms — Etsy penalizes this
- Use the pipe character `|` to separate logical sections for readability

**Examples:**

Minimalist: `French Bulldog Shirt Funny | Frenchie Coffee Lover Tee | Dog Mom Gift | Dogs and Coffee`

Photorealistic: `German Shepherd T-Shirt Dog Mom | Coffee and My GSD | Realistic Art Dog Lover Gift`

Line Art: `Black Lab Shirt | Black Coffee Black Lab No Small Talk | Dog Dad Gift Funny Dog Tee`

## 2. Pricing

Three tiers based on design style. These reflect the premium for complexity and
print quality. Always use Bella+Canvas 3001 as the base shirt.

| Style | Price | Rationale |
|---|---|---|
| Minimalist | $26.99 | Text-forward, fastest to produce, broad appeal |
| Line Art | $29.99 | Illustrated design adds perceived value |
| Photorealistic | $34.99 | Premium tier, hardest to copy, highest perceived value |

Never price below $26 for any style. Margins collapse below that after Etsy
fees and Printify production costs.

## 3. Tags

Etsy allows 13 tags, each up to 20 characters. Tags are multi-word phrases,
not single words. Every tag should be something a real buyer would search.

**Tag strategy — use all 13:**
- 3 breed-specific tags
- 2 saying/humor tags
- 2 gift occasion tags
- 2 "dog mom/dad/lover" tags
- 2 style/design tags
- 1 coffee angle tag
- 1 brand/niche tag

## 4. Description

Etsy descriptions should be 150–300 words. The first two sentences are shown
in search previews — make them count. Write in second person ("your dog,"
"you'll love"). Descriptions are not indexed for Etsy search, but they convert
browsers into buyers.

**Description template:**

```
[HOOK — 1-2 sentences using the saying or brand voice]

[BREED IDENTITY — 2-3 sentences about what it means to own this specific breed]

[PRODUCT DETAILS]
✦ Style: [Minimalist / Line Art / Photorealistic]
✦ Base shirt: Bella+Canvas 3001 Unisex Jersey Short Sleeve Tee
✦ Fabric: 100% Airlume combed and ring-spun cotton
✦ Fit: Unisex — runs true to size. Size up for a relaxed fit.
✦ Print: High-quality direct-to-garment (DTG) print that lasts wash after wash
✦ Colors available: [list colors]

[GIFT ANGLE — 1-2 sentences]

[CARE INSTRUCTIONS]
Machine wash cold, inside out. Tumble dry low. Do not iron on print.

[CLOSING — 1 sentence with brand name]
```

```
"Frenchie. Coffee. Silence." — if that's your holy trinity, this shirt was
made for you.

Being a French Bulldog owner means accepting the snoring, defending the vet
bills, and loving every dramatic, wrinkle-faced moment of it. This shirt gets
it. You wear it proudly.

✦ Style: Minimalist — bold text design, no fuss
✦ Base shirt: Bella+Canvas 3001 Unisex Jersey Short Sleeve Tee
✦ Fabric: 100% Airlume combed and ring-spun cotton
✦ Fit: Unisex — runs true to size. Size up for a relaxed fit.
✦ Print: High-quality DTG print built to last
```

## Batch Listing Requests

When asked to create listings for multiple breeds or the full catalog, produce
them in this order:
1. All minimalist listings first (fastest to list)
2. All line art listings
3. All photorealistic listings

Within each style, go breed by breed in this order: French Bulldog, German
Shepherd, Yellow Labrador, Black Labrador, Rottweiler, Long Hair Chihuahua,
Short Hair Chihuahua. (Ordered by Etsy search volume, highest first.)

## Quality Check Before Delivering

Before presenting any listing:
- Title is under 140 characters
- Title leads with breed name
- Exactly 13 tags, each under 20 characters
- Description opens with the saying or a hook — not with "This shirt..."
- Price matches the correct tier for the design style
- No generic "dog shirt" phrasing where breed-specific language belongs
