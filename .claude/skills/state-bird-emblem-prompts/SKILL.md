# State Bird Emblem AI Prompts

## Purpose

Aspect Media Holdings, LLC operates a State Bird Emblems product line: a 50-state
series of photorealistic bird emblem designs for stickers, patches, and t-shirts, plus
veteran-branch variants for Army, Navy, Air Force, Marines, Coast Guard, and Space Force.
This skill governs all AI image prompt generation for this product line, ensuring
consistent composition, style, and output across all 50 states and six veteran variants.

## When to Use

- Generating ImageFX, Midjourney, or DALL-E prompts for any state bird emblem
- Producing veteran-branch bird emblem prompts
- Building master prompt templates or per-state prompt reference documents
- Iterating on composition or style for the emblem series
- Briefing Meghan on design deliverables for this product line

## Inputs

Confirm before starting:
1. Scope: single state, batch of states, veteran variants, or full 50-state set
2. State name(s) and official state bird(s) — verify before prompting
3. State flag colors for the target state(s) — use official flag palette
4. Tool target: Google ImageFX (primary), Midjourney, or DALL-E
5. Whether this is a new prompt or a refinement of an existing one

For veteran variants, confirm branch: Army, Navy, Air Force, Marines, Coast Guard, Space Force.

## Procedure

### Standard Composition (apply to every emblem)

Every prompt must specify all of the following. Never omit an element.

- **Style**: photorealistic wildlife photography quality — not illustration, not vector
- **Pose**: heroic frontal pose, wings spread upward touching the circular frame boundary
- **Head**: turned slightly to the right, beak open in calling or display pose
- **Feet**: visible, with talons shown — do not allow cropping at feet
- **Frame**: circular emblem format, bird fills the frame
- **Colors**: state flag color palette applied to background and frame elements
- **Lighting**: dramatic natural lighting, sharp feather detail
- **Background**: complementary to flag colors, not distracting

### Prompt Structure

Build prompts in this order:
1. Subject: "[State Bird common name], [scientific name]"
2. Pose: "heroic frontal pose, wings spread upward to frame edges, head turned slightly right, beak open"
3. Feet: "full body visible, feet and talons clearly shown"
4. Style: "photorealistic wildlife photography, sharp feather detail, dramatic natural lighting"
5. Frame: "circular emblem composition, bird fills frame"
6. Colors: "[State] flag colors: [list primary colors] in background and border"
7. Negative (ImageFX): exclude illustration, cartoon, vector, cropped feet, blurry

### Per-State Reference Document

When producing a full 50-state document:
1. Table format: State | Official Bird | Common Name | Flag Primary Colors | Flag Secondary Colors
2. Verify official bird names — do not guess from memory
3. Include a copy-paste prompt block for each state
4. Flag any states where the official bird is shared with another state

### Output Batching

For batches larger than 10 states:
1. Group by region (Northeast, Southeast, Midwest, Southwest, West, Pacific)
2. Deliver one region at a time
3. Produce a master index with state, bird, and prompt status (draft / reviewed / final)

## Output Format

- Per-state prompts: copy-paste blocks, one per state, labeled clearly
- Master template: Word document (.docx) with table + prompt blocks
- HTML visual reference: optional, for previewing composition guidance
- Veteran variant set: six prompts, one per branch, with branch color specs

All prompts must be self-contained — no reference to external files or memory needed to use them.

## Quality Checks

Before delivering any prompt set:
- Confirm feet/talons visible is specified in every prompt.
- Confirm circular frame composition is specified.
- Confirm style is photorealistic, not illustration or vector.
- Confirm state flag colors are correctly attributed to the right state.
- Confirm veteran branch colors match branch standards.
- If producing a Word document, confirm it is formatted for Meghan's use — clear headers, copy-paste ready.
