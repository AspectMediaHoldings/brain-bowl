# Family Matching Shirts POD

## Purpose

Aspect Media Holdings, LLC operates a family matching t-shirt line on Etsy and Printify
targeting families at theme parks and entertainment venues. This skill governs product
strategy, trademark-safe design collection planning, AI image prompt generation, provider
selection, and Etsy listing production for this line. Meghan executes the design work.

## When to Use

- Planning new design collections for the family matching shirts line
- Generating AI image prompts for matching shirt graphic designs
- Selecting or comparing Printify providers for this product category
- Drafting Etsy listing titles, descriptions, or tags for matching shirts
- Reviewing any design concept or copy for trademark risk
- Producing pricing analysis or margin calculations for this line

## Inputs

Confirm before starting:
1. Task type: design planning, prompt generation, provider selection, listing copy, or trademark review
2. Target occasion: theme park, beach vacation, holiday, sports event, general family
3. Number of shirt variants in the set (individual, couples, parent/child, full family)
4. Whether Meghan has been briefed or if this is pre-brief planning

## Procedure

### Trademark Safety — Hard Rules (non-negotiable)

These rules apply to every design, listing, and prompt. No exceptions.

- Never reference Disney, Universal Studios, SeaWorld, Busch Gardens, or any branded park by name.
- Never use character names, character imagery, or any visual inspired by copyrighted characters.
- Never use phrases associated with specific park rides, attractions, or branded experiences.
- Mickey Mouse ears silhouette: do not use even as a "generic" shape — it is protected.
- If a design concept requires naming or depicting a specific brand to work: reject it and substitute a generic version.
- When in doubt, do not proceed — flag for trademark review first.

Generic is not a workaround for a trademark concept. Designs must stand on their own without the brand association.

### Design Collections

Eight approved generic collections (trademark-safe):

1. **Theme Park Family** — roller coasters, ferris wheels, funnel cake, generic park icons
2. **Beach Vacation Crew** — waves, sandcastles, sunsets, palm trees
3. **Mountain Adventure Squad** — hiking boots, pine trees, peaks, campfire
4. **Road Trip Family** — vintage car, map, highway signs, open road
5. **Camping Crew** — tent, lantern, s'mores, stars
6. **Sports Family** — generic jersey numbers, pennants, stadium silhouette
7. **Holiday Matching** — seasonal and holiday-specific, rotate by occasion
8. **Foodie Family** — pizza, tacos, ice cream, diner icons

When developing a new collection outside these eight: run trademark review before proceeding.

### AI Image Prompt Generation

For each design:
1. State the collection name and occasion
2. Describe the graphic: subject, composition, style
3. Specify style: flat vector illustration, retro vintage, hand-drawn, or modern minimalist
4. Specify color palette: family-friendly, bright, or specific brand palette for the shop
5. State the text treatment if any: font style, what the text says, placement
6. Negative: no branded characters, no park names, no recognizable IP

Prompt structure:
"[Style] illustration of [subject], [composition], [color palette], [text treatment if any], suitable for family matching t-shirt, commercial print-on-demand use"

### Provider Selection

| Provider | Fulfillment | Cost | Best For |
|---|---|---|---|
| Printify | 7-14 days | Budget | Standard orders, margin priority |
| SPOD | 5-7 days | Mid | Speed priority, still margin-positive |
| Printful | 6-8 days | Premium | Quality priority, premium positioning |

Default to Printify for this line unless speed or quality is the stated priority.
Confirm provider with Nathan before publishing listings.

### Pricing

Target: 35-50% profit margin. $28-40 per family set order.
Calculate per-variant cost, add Etsy fees (6.5% transaction + listing), then set retail price to hit margin target.
Bundle pricing: offer family set discount (e.g., buy 4 save 10%) to increase average order value.

### Etsy Listing Production

1. Title: lead with occasion (e.g., "Family Vacation Matching Shirts"), include size range and material.
2. Description: open with the value statement (matching, coordinated, easy to order together). Include size chart, processing time disclosure, and care instructions.
3. Tags: 13 tags maximum. Mix: occasion, size descriptors, "matching family shirts," "vacation outfit," "family photos," specific occasion words.
4. Photos: lifestyle first (family wearing shirts), flat lay second, size chart third.

## Output Format

- Design briefs: Word document with collection name, occasion, design description, prompt, and color specs — formatted for Meghan handoff.
- Prompts: copy-paste blocks, one per design, labeled by collection and variant.
- Listing copy: titled sections per Etsy field (Title, Description, Tags), ready to paste.
- Provider comparison: table in Word or PDF.
- Pricing analysis: table with cost, Etsy fees, retail, and margin per variant.

## Quality Checks

Before delivering any design brief or prompt:
- Confirm no brand names, character references, or protected imagery are present.
- Confirm the generic concept works without the brand association.

Before delivering listing copy:
- Confirm trademark-safe language throughout.
- Confirm processing time disclosure matches current Printify estimates.
- Confirm 13 tags are used (Etsy maximum).

Before delivering pricing:
- Confirm margin is at or above 35% after all fees.
- Confirm Etsy transaction fee (6.5%) and listing fee are included in the calculation.
